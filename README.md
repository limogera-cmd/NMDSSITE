# NMDS — Nemodes | Site Oficial

Site oficial da banda **NMDS (Nemodes)** — rock brasileiro.

## ▶️ Como rodar

Basta abrir o `index.html` no navegador, ou usar um servidor local como o Live Server do VSCode.

## 📁 Imagens necessárias

Coloque as imagens na pasta `assets/images/` com esses nomes:

| Arquivo                    | Uso                            |
|----------------------------|--------------------------------|
| `band-photo-1.jpg`         | Foto hero (background da capa) |
| `band-photo-2.jpg`         | Foto da seção "Sobre"          |
| `logo-nmds.png`            | Logo principal (fundo transparente) |
